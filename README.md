# 👕 Emprende Ropa - Tienda Virtual

Tienda virtual de ropa desarrollada con HTML, CSS y JavaScript puro. Permite a los usuarios explorar productos, agregar al carrito y realizar compras simuladas.

## 🚀 Demo

🔗 **Sitio en vivo:** [https://tu-usuario.github.io/emprende-ropa](https://tu-usuario.github.io/emprende-ropa)

## ✨ Características

- ✅ Registro e inicio de sesión de usuarios
- ✅ Catálogo de productos (hombre y mujer)
- ✅ Carrito de compras funcional
- ✅ Ajuste de cantidades en el carrito
- ✅ Eliminación de productos del carrito
- ✅ Proceso de checkout simulado
- ✅ Diseño responsive
- ✅ Persistencia de datos con localStorage
